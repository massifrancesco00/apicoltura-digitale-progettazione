# Progettazione Funzioni 3.1/3.2/3.4 

1. >**Dopo aver premuto il pulsante del menù a tendina detto anche pulsante ad Hamburger si aprirà il suddetto menù e mostreà le opzioni Logout/Cambia Apiari/Impostazioni Arnia.**
    ```javascript
    function OpenMenuTendina();
    ```

2. >**Premuta l'opzione Logout dovrà aprirsi la schermata di Login in modo da poter accedere di nuovo** 
    ```javascript
    function OpenFormLogin();
    ```
  3. >**Premuta l'opzione cambia apiari dovrà aprirsi la schermata per la scelta degli apiari che si posseggono per poi aprire la schermata giusta**
       ```javascript
       function OpenFormSceltaApiari();
       ```
   4. >**Premuta l'opzione Impostazioni Arnia dovrà aprirsi la schermata delle impostazioni delle soglie dell'arnia**
          ```javascript
          function openFormImpostazioniArnia();
          ```
